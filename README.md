## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/qatar-stadiums.git
```

Go to the project directory

```bash
  cd qatar-stadiums
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
